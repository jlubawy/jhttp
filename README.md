# jhttp

Very simple example of an HTTP server written in Java using just Sockets.

## Build

Windows:

   > build.bat

Unix/Mac:

   > ./build.sh

## Run

Windows:

   > run.bat [TYPE]

Unix/Mac:

   > ./run.sh [TYPE]

Where [TYPE] is either 'html', 'json', or 'text'. Open your browser to [http://localhost](http://localhost).
